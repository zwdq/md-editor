# ✏️ Markdown 在线编辑器

Typora 风格的 Markdown 编辑器，纯前端实现，无需后端。

## 功能

- ✍️ **实时渲染** — 输入即渲染，所见即所得
- 📐 **三种模式** — 编辑 / 分屏 / 预览，可拖拽调整比例
- 🌙 **深色/浅色主题** — 一键切换
- 💾 **自动保存** — 内容自动存 localStorage，刷新不丢失
- 📥 **打开文件** — 支持 .md/.txt/.markdown
- 📤 **下载** — 导出为 .md 文件
- 📋 **复制 HTML** — 一键复制完整 HTML
- 📋 **模板** — 内置 Markdown 语法模板
- ⌨️ **Tab 缩进** — 支持 Tab 键缩进
- 📱 **响应式** — 手机端上下分屏

## Markdown 支持

- 标题 H1-H6
- 粗体、斜体、删除线
- 代码块（```lang）、行内代码
- 引用块
- 有序/无序列表、任务列表
- 表格
- 链接、图片
- 水平线

## 技术

纯 HTML/CSS/JS，零依赖，部署在 Cloudflare Pages。
